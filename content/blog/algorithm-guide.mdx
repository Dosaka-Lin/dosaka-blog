### 核心算法路线

1. **基础数据结构**：数组、链表、栈、队列、哈希表
2. **基础算法**：排序、二分、双指针、前缀和
3. **搜索**：DFS、BFS、回溯
4. **动态规划**：线性DP、区间DP、树形DP
5. **图论**：最短路、最小生成树、拓扑排序
6. **数学**：数论、组合数学、博弈论

### 刷题策略

- 🔴 **Codeforces**：参加每场 Div.2 / Div.3 比赛
- 🟣 **Luogu**：按题单刷题，从普及- 到提高+
- 🟠 **牛客**：刷校招真题，准备笔试

> 坚持每天至少一道题，保持手感。

### 常用模板

```cpp
// 快速幂
long long qpow(long long a, long long b, long long mod) {
    long long res = 1;
    while (b) {
        if (b & 1) res = res * a % mod;
        a = a * a % mod;
        b >>= 1;
    }
    return res;
}
```
